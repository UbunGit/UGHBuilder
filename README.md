# UGHBuilder
快速简易集成HBuilder
## 简要
在集成HBuilder的时候，走了很多弯路，踩了很多坑，为了之后方便快速集成，所以将集成修改为pods方式
## 怎么使用
pod 'UGHBuilder', '~> 1.0.0'
## 包含内容

### HBuilder静态文件
```
libcoreSupport.a
liblibNativeObj.a
liblibNativeUI.a
liblibNavigator.a
liblibPDRCore.a
liblibStorage.a
liblibUI.a
```
### Frameworks
```
storage.framework
```
